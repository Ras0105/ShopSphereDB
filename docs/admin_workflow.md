# ShopSphere – Admin Workflow

```text
Admin opens Admin Panel
        ↓
Admin Login
        ↓
Dashboard
        ↓
Manage Categories
(Add / Update / Delete Categories)
        ↓
Manage Products
(Add / Update / Delete Products)
        ↓
Manage Inventory
(Update Stock / Reorder Stock)
        ↓
Manage Orders
(View New Orders)
        ↓
Verify Payment Status
        ↓
Process Orders
        ↓
Update Order Status
Pending
   ↓
Packed
   ↓
Shipped
   ↓
Out for Delivery
   ↓
Delivered
        ↓
Handle Returns/Refunds (Optional)
        ↓
View Reviews & Customer Feedback
        ↓
Generate Reports & Analytics
(Sales, Revenue, Low Stock, Top Products)
```
